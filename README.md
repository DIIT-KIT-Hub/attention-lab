# AttentionLab

Веб-платформа для дослідження візуального сприйняття та залучення уваги користувачів за допомогою eye tracking.

## Про проект

**Мета:** створити інструмент для дослідження того, як користувачі сприймають візуальну інформацію, що привертає увагу, в якому порядку, як довго.

**Компоненти:**
1. **Eye Tracking модуль** - відстеження погляду через веб-камеру (WebGazer.js)
2. **Генератор шаблонів** - створення візуальних стимулів з різними засобами залучення уваги
3. **Опитувальник** - збір суб'єктивних відповідей користувача
4. **Аналізатор** - обробка та візуалізація результатів

## Технології

- **Frontend:** React
- **Eye Tracking:** WebGazer.js
- **Візуалізація:** Canvas API
- **Стилізація:** Tailwind CSS
